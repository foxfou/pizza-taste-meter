# Pizza Taste Meter

Шуточный проект для моих друзей, где можно оценить вкус моей пиццы.

## Что это?

Это простое веб-приложение, которое позволяет друзьям оставлять оценки и отзывы о пицце, которую я готовлю. Никаких серьёзных целей — просто веселье и честная обратная связь!

## Как запустить

```bash
npm install
npm run dev
```

Приложение будет доступно по адресу `https://pizza-taste-meter.netlify.app/`.

## Технологии

- React Router
- TypeScript
- TailwindCSS
